An open-source library for automated HTTP Tests. Everything you need to know to get started is displayed in the code example below. By the way, restfuse also supports the testing of asynchronous services. 

@RunWith( HttpJUnitRunner.class )
public class RestfuseTest {

  @Rule
  public Destination destination = new Destination( "http://restfuse.com" ); 

  @Context
  private Response response;
		
  @HttpTest( method = Method.GET, path = "/" )
  public void checkRestfuseOnlineStatus() {
    assertOk( response );
  }  
}
