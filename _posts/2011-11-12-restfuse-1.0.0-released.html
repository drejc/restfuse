--- layout: post title: Restfuse 1.0.0 released --- We are proud to announce the 1.0.0 release of resfuse. This release marks also the start of this poject. The 1.0.0 version contains the following features:
  • The @HttpTest annotation to tell restfuse to send a http request before executing the annotated method as a test method.
  • The @Poll and @Callback annotation to handle asynchronous services.
  • Injection of responses into the test object.
  • A utility class called com.eclipsesource.restfuse.Assert to ease the testing of response codes.
  • Basic and Digest Authentication support.
  • A JUnitRunner called HttpJUnitRunner that detects @HttpTest annotated methods as test methods.
  • Support for twelve Media-Types.
There is also a detailed documentation and a first version of a FAQ. Restfuse can be downloaded as an archive or consumed via Maven and p2.
